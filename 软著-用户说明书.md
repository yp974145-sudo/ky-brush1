# 考研公共课刷题平台 V1.0 — 用户说明书

---

## 一、引言

### 1.1 编写目的

本文档是「考研公共课刷题平台」（以下简称"本软件"）的用户操作说明书，旨在帮助用户快速了解软件功能、掌握操作方法，充分发挥软件的刷题备考价值。

### 1.2 软件概述

本软件是一款面向全国硕士研究生入学考试考生的在线刷题工具，为纯前端 Web 应用，无需安装，在浏览器中即可运行。覆盖**408 计算机学科专业基础综合**、**数学（一/二/三）**、**政治**、**英语（一/二）** 四大公共课门类，收录历年真题数千道，支持单选、多选、填空三种题型，提供即时判题、错题重做、模拟考试、学习规划等全方位备考功能。

### 1.3 软件特点

- 📚 **多学科覆盖**：408 计算机四科（数据结构、组成原理、操作系统、网络）+ 数学 + 政治 + 英语
- 📅 **年份筛选**：真题年份跨度从 1986 年至 2025 年
- ✅ **即时判题**：选择答案后立即显示对错与解析
- 📋 **答题卡**：可视化展示已答/未答/正确/错误/标记状态
- 🔁 **错题本**：自动收集错题，支持重做与移出
- ⏱ **模拟考试**：可配置科目、题数、时长，模拟真实考场
- ⌨️ **键盘快捷键**：ABCD 选题，方向键切题，回车提交
- 💾 **本地存储**：答题进度自动保存，刷新页面不丢失
- 🌓 **深色模式**：护眼深色主题，适合长时间刷题
- 👥 **多账户**：一台设备支持多用户切换，进度隔离
- 🔍 **全文搜索**：支持按题目内容、知识点关键字搜索
- 💻 **代码草稿板**：内置 JavaScript / C 语言在线运行环境

## 二、运行环境

### 2.1 硬件环境

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| 处理器 | 任意双核 CPU | 2.0GHz 以上 |
| 内存 | 2 GB | 4 GB 以上 |
| 存储 | 10 MB 可用空间 | — |
| 网络 | 首次加载需联网 | 支持离线使用（PWA） |
| 显示 | 分辨率 ≥ 360×640 | 1920×1080 |

### 2.2 软件环境

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows / macOS / Linux / Android / iOS |
| 浏览器 | Chrome 80+ / Edge 80+ / Safari 14+ / Firefox 80+ |
| 其他 | 无需安装任何插件或运行时 |

### 2.3 部署方式

本软件为纯静态 Web 应用，部署极其简便：

- **方式一**：直接用浏览器打开 `index.html`
- **方式二**：使用任意 HTTP 服务器托管静态文件目录
  ```bash
  python -m http.server 8080 --directory .
  ```
- **方式三**：部署到 GitHub Pages、Vercel、Netlify 等静态托管平台

## 三、功能操作说明

### 3.1 主界面概览

打开软件后进入主界面，页面布局如下：

- **顶部导航栏**：显示 Logo、统计数据（总题数/正确/错误/正确率）、功能按钮（考试、错题、登录、更多）
- **左侧筛选栏**：科目组、学科、知识点、年份、题型筛选器，以及快捷模式按钮
- **中央内容区**：题目展示区，作答区和结果反馈区
- **右侧答题卡**（桌面端）/ 底部滑出面板（移动端）：题目完成状态概览

> 📸 *截图1：主界面全貌*

### 3.2 选择科目与筛选题目

用户可以通过左侧筛选栏精确选择要练习的题目范围：

1. **科目组筛选**：按公共课大类快速切换（408计算机 / 数学 / 政治 / 英语）
2. **学科筛选**：选择具体学科，如"数据结构"、"高等数学"
3. **知识点筛选**：选择单个学科后，可进一步筛选具体知识点，如"二叉树"、"极限与连续"
4. **年份筛选**：选择特定年份的真题
5. **题型筛选**：单选 / 多选 / 填空

选中筛选项后，中央区域自动更新题目列表，底部"当前筛选"面板显示匹配题目数量与完成情况。

> 📸 *截图2：科目与年份筛选*

### 3.3 答题与判题

在题目展示区，用户可进行如下操作：

1. **单选题**：点击选项字母（A/B/C/D）或按键盘对应字母键选中，点击"提交"或按回车确认
2. **多选题**：依次点击多个选项，再次点击取消选择，提交后系统比对答案
3. **填空题**：在输入框中输入答案，按回车提交

提交后系统立即判题：
- ✅ 正确：显示绿色反馈，附答案解析
- ❌ 错误：显示红色反馈，自动收入错题本，附正确答案与解析

> 📸 *截图3：答题正确反馈*

> 📸 *截图4：答题错误反馈*

### 3.4 答题卡

答题卡以网格形式展示当前筛选范围内所有题目的作答状态：

- ⚪ 灰色圆点：未作答
- 🟢 绿色圆点：回答正确
- 🔴 红色圆点：回答错误
- 🟠 橙色圆点：已标记

点击答题卡中的任意题号可快速跳转到该题。桌面端答题卡固定在右侧，移动端通过底部"答题卡"按钮滑出。

> 📸 *截图5：答题卡面板*

### 3.5 错题本与错题重做

系统自动收集所有答错的题目进入错题本：

- **浏览错题**：点击侧栏「浏览错题」或顶部「错题」按钮，列出所有错题
- **错题重做**：从错题本中随机抽取指定数量的题目进行重做，答对自动移出错题本
- **清空错题**：在"更多"菜单中可重置错题本

> 📸 *截图6：错题重做完成统计*

### 3.6 收藏功能

对于需要反复练习或重点关注的题目，可点击题目下方的「☆ 收藏」按钮加入收藏夹。收藏的题目可在侧栏「⭐ 我的收藏」中集中查看。

### 3.7 个人笔记

每道题目下方设有笔记区域，用户可记录解题思路、易错点等个性化内容。笔记自动保存，跟随题目永久存储。

> 📸 *截图7：个人笔记区域*

### 3.8 模拟考试

模拟考试功能还原真实考场体验：

1. 点击「⏱ 考试」按钮进入考试设置面板
2. 配置考试参数：
   - **科目**：选择单科或全部随机
   - **题数**：5~100 题可调（滑块选择）
   - **时长**：30 / 45 / 60 / 90 / 120 / 180 分钟可选
3. 点击「🚀 开始考试」进入考试模式
4. 考试期间顶部显示倒计时，倒计时结束自动交卷
5. 交卷后展示成绩报告：正确率、正确/错误题数、用时，以及逐题回看

> 📸 *截图8：模拟考试设置*

> 📸 *截图9：考试成绩报告*

### 3.9 随机刷题

点击侧栏「🎲 随机刷题」按钮，系统将所有题目随机打乱顺序，适合碎片时间练习。

### 3.10 全文搜索

顶部搜索栏支持实时搜索：

- 输入关键字后，题目列表实时过滤，匹配题目中的文字和知识点名称
- 匹配关键字在题目中高亮显示
- 搜索范围自动限定在当前筛选结果内

> 📸 *截图10：搜索结果高亮*

### 3.11 代码草稿板

点击「💻 代码草稿板」按钮，从底部滑出在线代码编辑器：

- 支持 **JavaScript** 和 **C 语言**两种语言
- 点击「▶ 运行」在线执行代码，输出结果显示在下方
- 适合 408 考生练习算法题、验证代码逻辑

> 📸 *截图11：代码草稿板*

### 3.12 多账户管理

在顶部「登录」按钮进入账户管理面板：

- **创建账户**：输入昵称即可创建独立答题账户
- **切换账户**：在同一设备上切换不同用户，数据完全隔离
- **数据导出/导入**：支持将答题数据导出为 JSON 文件备份，或从备份导入

> 📸 *截图12：多账户管理面板*

### 3.13 学习统计

「📊 统计」面板展示个人学习数据：

- 各科目完成题数、正确率
- 每日打卡记录与连续打卡天数
- 答题趋势可视化

### 3.14 学习规划器

「📋 学习规划器」帮助考生制定系统化复习计划：

- 按轮次规划各科目复习进度
- 设置每日题量目标
- 跟踪完成百分比

### 3.15 题库管理

「📦 题库管理」允许用户自定义扩充题库：

- 手动添加新题目（含题干、选项、答案、解析）
- 导入/导出题目数据

### 3.16 深色模式

通过「更多 → 🌓 深色模式」切换深色主题，降低长时间刷题的眼部疲劳。

### 3.17 考研指南

「📅 考研指南」提供考研相关实用信息：

- 报名时间节点
- 考试日期倒计时
- 各阶段备考建议

### 3.18 分享功能

「📤 分享」可将当前学习成果生成精美卡片，分享至社交平台。

### 3.19 PWA 离线支持

本软件支持 PWA（渐进式 Web 应用），在支持的浏览器中可添加到主屏幕，离线状态下仍可使用。

## 五、数据存储说明

本软件使用浏览器 localStorage 存储所有用户数据，包括：

- 答题记录与进度
- 错题本
- 收藏夹
- 个人笔记
- 打卡记录
- 考试历史

数据完全存储在用户本地设备，不会上传至任何服务器（除非用户主动使用云同步功能）。清除浏览器数据会导致答题记录丢失，建议定期使用"数据导出"功能备份。

## 六、常见问题

**Q：换设备后数据还在吗？**
A：本软件数据存储在浏览器本地，换设备需先导出数据（JSON文件），再在新设备导入。

**Q：题目来源是什么？**
A：题目来自公开的历年考研真题，仅供学习参考使用。

**Q：能否离线使用？**
A：可以。首次加载后，浏览器会缓存所有资源文件，后续即使无网络也能正常刷题。

**Q：如何在手机上使用？**
A：直接在手机浏览器中打开即可，界面已做移动端适配。支持添加到手机主屏幕作为独立 App 使用。
