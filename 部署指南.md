# 🎯 考研刷题 - 部署指南

纯静态网站，扔到任意服务器就能用。推荐 GitHub Pages（免费，HTTPS，全球CDN）。

---

## 方式一：GitHub Pages（推荐，手机电脑都能用）

### 1. 创建 GitHub 仓库

打开 https://github.com/new

- Repository name 填 `ky-brush`（或任意名字）
- 选 **Public**
- 不要勾选 "Add a README file"
- 点 **Create repository**

### 2. 推送代码

在 `E:\hanako\408-brush` 目录下打开终端（PowerShell/Git Bash），逐行执行：

```bash
git init
git add .
git commit -m "考研刷题 v1.0"
git branch -M main
git remote add origin https://github.com/你的用户名/ky-brush.git
git push -u origin main
```

> 把 `你的用户名` 替换成你的 GitHub 用户名。执行后会弹出 GitHub 登录窗口。

### 3. 开启 Pages

- 进仓库 → **Settings** → 左侧 **Pages**
- **Branch** 选 `main`，文件夹选 `/ (root)`，点 **Save**
- 等1-2分钟，上方会出现网址：`https://你的用户名.github.io/ky-brush`

### 4. 完成 ✅

用手机浏览器打开 `https://你的用户名.github.io/ky-brush`，就能刷题了。

## 方式二：本地直接用

双击 `index.html`，浏览器打开就能用。缺点是只能在这台电脑上访问。

---

## 方式三：自己的服务器

整个 `408-brush` 文件夹扔到任意静态文件服务器（Nginx/Apache/OSS）即可，零依赖。

## 更新题目

编辑 `js/` 下的数据文件，按格式加题即可。推送到 GitHub 后 Pages 自动更新。

```js
// data-408.js / data-math.js / data-politics.js / data-english.js
{
  id: '年份-序号-科目',    // 唯一ID，如 '2024-05-os'
  year: 2024,             // 年份
  subject: 'os',          // 科目代码
  topic: 'os-2',          // 知识点代码
  type: 'single',         // single | multi | fill | essay
  difficulty: 'medium',   // easy | medium | hard
  question: '题目描述...',
  code: '代码块（可选）',  // 可选
  passage: '阅读原文（英语）', // 可选
  options: ['A. ...', 'B. ...', 'C. ...', 'D. ...'],
  answer: 'A',            // 单选:'A' 多选:['A','B','C'] 填空:'答案文本'
  analysis: '解析...'
}
```
