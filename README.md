# 408 计算机考研刷题

仿牛客网的在线刷题平台，覆盖 408 四大学科全真历年真题。

## 功能

- 📚 四大学科：数据结构、计算机组成原理、操作系统、计算机网络
- 📅 年份筛选：2009-2025 年真题
- ✅ 即时判题：选择答案后立即显示对错和解析
- 📋 答题卡：一目了然看到已答/未答/正确/错误/标记
- 🔁 错题本：自动收集错题，支持重做
- 🎲 随机刷题：打乱顺序练习
- ⌨️ 键盘快捷键：ABCD 选题，方向键切题，回车提交
- 💾 本地存储：进度自动保存，刷新不丢

## 部署

纯前端静态页面，无需后端。扔到任何静态服务器即可：

```bash
# Python
python -m http.server 8080 --directory .

# 或直接用浏览器打开 index.html
```

## 添加题目

编辑 `js/data.js`，在 `QUESTION_BANK` 数组中按格式添加：

```js
{
  id: '2009-01-ds',         // 唯一ID
  year: 2009,               // 年份
  subject: 'ds',            // ds/co/os/cn
  type: 'single',           // 题型
  question: '题目描述...',
  code: '代码块（可选）',
  options: ['A. ...', 'B. ...', 'C. ...', 'D. ...'],
  answer: 'A',              // 正确答案
  analysis: '解析...'       // 解析（可选）
}
```
